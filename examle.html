CheckBrowser JS

An easy way to warn user about deprecated browser.

Get CheckBrowser JS on GitHub.

Features

  • Unobtrusive (shows message once per visit)
  • Browser detection (checkBrowser.getBrowser() returns an array with browser name and version)
  • Multilanguage (with automatic user language detection)
  • Customizable (modify css/checkbrowser.css)

Usage

  • Insert checkbrowser.js (<script src="checkbrowser.js"></script>).
  • Call checkBrowser.warning() after page load (window.onload = function() { checkBrowser.warning(); }).

Optional

  • Modify styles in checkbrowser.css
  • Spicify config in checkBrowser.warning() (window.onload = function() { checkBrowser.warning({language:'en', css:'css/checkbrowser.css'}); })
  • Modify defaults in checkbrowser.js

Config options

  • cookie — cookie name
  • css — path to css file
  • language — user language
  • messages — warning messages array
  • modern — modern browsers
  • browsers — show this browsers in warning
