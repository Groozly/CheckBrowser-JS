CheckBrowser JS

An easy way to warn user about deprecated browser.

Get CheckBrowser JS on GitHub.

Usage

  • Insert checkbrowser.js (<script src="checkbrowser.js"></script>).
  • Call checkBrowser.warning() after page load (window.onload = function() { checkBrowser.warning(); }).

Optional

  • Modify styles in checkbrowser.css
  • Modify defaults in checkbrowser.js (or modify before checkBrowser.warning())
